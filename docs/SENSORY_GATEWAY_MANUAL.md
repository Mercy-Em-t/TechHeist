## The Execution Process

When you pass a raw sentence into the Gateway via the `send_text_directive` tool, the following biological sequence occurs:

1. **Ingestion (`sensory-gateway.ts`)**: The module parses your text and runs natural language intent mapping.
2. **Classification**:
   - *Minting Intent*: If the sentence includes words like "mint", "build", or "launch", it dynamically invokes the DNA Sequencer to trigger cellular mitosis and spawn a new SaaS/DevOps branch cluster tailored to your request.
   - *Diagnostic Intent*: If the sentence implies checking, fixing, repairing, or status verification, the Gateway initiates a `StateMachineEngine` sequence to run a background diagnostic (e.g., `npm run build`) and self-heal any syntax anomalies.
   - *Global Broadcast*: For everything else, it secretes a `HUMAN_CONVERSATIONAL_INJECTION` hormone directly into the bloodstream, distributing your raw message to all listening sibling nodes to adapt.
3. **Execution**: The target cells catch the hormone and autonomously execute their specialized behaviors (e.g., the `STEM_SAAS` cell synthesizing the React code or the `ROOT_DEVSECOPS` pushing to the edge).
4. **Graph Integration**: Every interaction is logged directly into the `network-state.json` graph map under the tag `sensory-input`, which immediately renders on your 3D Visual Command Dashboard (`http://localhost:8081`).
