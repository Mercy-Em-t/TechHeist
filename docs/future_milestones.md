# Future Milestones

The following ideas have been temporarily sidelined to focus on current priorities, but should be revisited later:

- **Intelligent Ingestion:** Integrating an LLM API to handle unstructured text intelligently, expanding the `BuilderAgent`'s parsing logic beyond simple rules.
- **Query & Retrieval:** Building a `QueryAgent` that can dynamically traverse the `GraphStore` to answer complex questions based on the interconnected context we've built.

### Core Vision Reminder
> **"Our end goal is autonomous systems; MCP is just one of the tools to get there."** 
> MCP acts as the universal USB port to rapidly plug into existing infrastructure without spending years building from scratch, but it must be wired with a 2035 autonomous mindset.
